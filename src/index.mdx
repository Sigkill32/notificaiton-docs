# Introduction

```notification``` component is a customizable react notification and message component clubbed together. ```notification``` allows you to spawn ether notification or message component just by altering one prop instead of using entirely different components.

## Features

- **All in one**: Spawn either a message or a notification component without having to use two separate components.
- **Stateless**: Spawn your notification component without having to worry about maintaining states.
- **customizable**: Customize components to fit your needs.
